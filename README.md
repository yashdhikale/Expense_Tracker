# Centra Logic Assignment 3
## Installation

Step 1: Clone the repository

```
 git clone https://github.com/yashdhikale/Expense_Tracker.git
```
 Step 2 : Change directory to Color-Game
```
 cd Expense-Tracker
```

Step 3 : Install Node Modules
```
 npm install
```

Step 4 : Run the app
```
 npm run dev
```
